# 当前局域网运行入口

本剧情增强版默认使用 `3002` 端口，避免和旧版 `3001` 服务冲突。

双击项目根目录中的：

```text
RUN_GAME_NOW_WINDOWS.bat
```

或：

```text
start_game_windows.bat
```

本机打开：

```text
http://127.0.0.1:3002/
```

同一 Wi-Fi / 同一局域网设备打开：

```text
http://10.253.36.14:3002/
```

手机访问时不要使用 `127.0.0.1`，必须使用上面的局域网地址。若无法打开，请确认电脑和手机在同一 Wi-Fi，且 Windows 防火墙允许 Node.js 访问专用网络。
